Navegadores Web

Mozilla Firefox

Navegador web libre y de código abierto​ desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla. Usa el motor Gecko para renderizar páginas web, el cual implementa actuales y futuros estándares web.

Esta comunidad está conformada por:

  • Pensadores.
  • Tecnólogos.
  • Desarrolladores.

Mozilla Developer Edition

Firefox Developer Edition sustituye al canal Aurora en el Proceso de lanzamiento de versiones de Firefox. En Developer Edition, como en Aurora, las caracteristicas se lanzan cada seis semanas, despues de que se hayan estabilizado en Nightly builds. Conoce más sobre Mozilla Manifiésto.
